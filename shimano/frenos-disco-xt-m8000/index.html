Caracteristicas de los Frenos para Bicicletas XT m8000 de Shimano para Bicicletas de Montaña

Gracias al sistema Rider Tuned los nuevos discos de frenos para bicicleta aumentaran la seguridad activa ya que te ayudarán a frenar sin perder el control.

Todo esto con el mínimo recorrido de la maneta y utilizando si quieres solo dos dedos.

La principal mejora con respecto a los anteriores frenos de esta marca ha sido hacer más ligero todo el sistema de frenado.

Cabe destacar que puedes comprar el Kit completo o si lo necesitaras las piezas sueltas.

Entre las principales mejoras hay que destacar la modificación del cilindro principal, ahora es más ligero y más resistente. Las manetas denominadas Servo Wave mejoran el tacto a la hora de controlar la frenada.

Estas manetas Servo Wave tiene diferentes tipos de regulación para que se ajuste perfectamente al tamaño de la mano.

Dentro del catalogo para ciclismo Shimano Xt es la serie dedicada el ciclismo de Montaña

Shimano ha fabricado estos frenos XT m800 para bicicletas de montaña con los que mejorar la frenada tanto de la rueda delantera como la trasera.

Esto unido a que el precio del kit completo no es muy elevado, podríamos decir que si te gusta el enduro mtb es una de las mejores compra para actualizar el nuevo sistema de frenos de tu bici.
